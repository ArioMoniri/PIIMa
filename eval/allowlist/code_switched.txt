# inflectional suffixes, exactly as they surface in clinical notes. This is the
# boundary where a token classifier confuses a suffixed medical term with a
# suffixed Turkish surname (`Behçet'li` vs `Ahmet'li`), so every category's
# terms may appear here.

# Vowel harmony means one suffix has several surface forms (-de/-da/-te/-ta,
# -li/-lı/-lu/-lü, -den/-dan/-ten/-tan). Variants are listed rather than one
# canonical form, because hardcoding a single variant misses the others.
# Apostrophe-less surface forms are included: writers omit the apostrophe.
#
# Format: one form per line, UTF-8, sorted by Unicode code point, '#' comments.
